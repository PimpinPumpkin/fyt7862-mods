# Wallpaper

The home wallpaper cloned from the 7870 onto the 7862.

| File | What |
|---|---|
| `wallpaper_7870_1536x2048.png` | Full‑res source (the 7870's original) — on device as `/sdcard/Pictures/wallpaper_from_7870.png` |
| `wallpaper_7870_768x1024.png` | Resized to the 7862 panel (768×1024) |

Apply via Settings → Wallpaper or Lawnchair's picker. No root needed — it's a normal wallpaper.
